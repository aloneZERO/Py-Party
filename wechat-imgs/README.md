### 使用方法（前提是设备安装了python3）：

下载本项目到本地，打开项目主目录，打开命令行，输入：

```python
pip install -r requirements.txt # 安装第三方依赖
```

等待安装完成，输入：

```python
python wechat-imgs.py
```

本地的图片查看软件会自动打开二维码（扫描后即自动删除）：

用手机微信右上角的扫一扫，确认登陆即可。

稍等片刻，你打开手机微信，找到信息栏的微信传输助手就会看到了。
